# 한국 배경 레퍼런스

## 기본 방향
현실 한국 + 별빛 판타지. 장소는 실제로 한국에 있을 것처럼 보이되, 브랜드 로고를 그대로 복제하지 않는다. 편의점은 7-Eleven, CU, GS25 같은 한국 편의점 문화에서 받은 느낌만 사용하고, 가상의 간판으로 만든다.

## 사용할 수 있는 장소
1. 밤 편의점 : 삼각김밥, 컵라면 온수기, 바나나우유, 교통카드 충전기, 우산꽂이.
2. 학교 앞 문구점 : 스티커, 샤프심, 뽑기 기계, 작은 장난감.
3. 아파트 놀이터 : 미끄럼틀, 모래, 자전거 거치대, 경비실 불빛.
4. 버스정류장 : 노선도, 전광판, 비 오는 유리, 교통카드 소리.
5. 지하철 입구 : 계단, 노란 점자블록, 바람, 광고판 빛.
6. 한강 산책로 : 자전거길, 라면 기계, 편의점 테라스, 강물 반사.
7. 시장 골목 : 천막, 과일 상자, 붕어빵, 어묵 국물 김.
8. 분식집 : 떡볶이 김, 순대 접시, 물컵, 빨간 의자.
9. 코인노래방 앞 : 네온, 작은 복도, 반짝이는 문.
10. 장마철 횡단보도 : 우산, 물웅덩이, 신호등 반사.
11. 카페 골목 : 작은 테이블, 따뜻한 창문빛, 골목 고양이 금지. 동물 캐릭터 추가 금지.
12. 옥상 빨래줄 : 바람, 도시 불빛, 작은 화분.
13. PC방 간판 거리 : 모니터 빛, 계단, 간판 반사. 게임 화면은 추상화.
14. 겨울 포장마차 : 붕어빵 틀, 호떡, 흰 입김, 노란 전구.
15. 편의점 뒷골목 : 분리수거함, 박스, 작은 별빛 틈.

## 금지
- 실제 브랜드 로고를 그대로 사용하지 않는다.
- 한 배경만 100편 반복하지 않는다.
- 한국 배경을 단순 장식으로만 쓰지 않는다. 사건의 규칙과 해결이 장소의 물건에서 나와야 한다.
