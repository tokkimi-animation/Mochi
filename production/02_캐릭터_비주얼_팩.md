# 캐릭터 비주얼 팩

## 공통 스타일
고급 3D 소프트 플러시 렌더. 둥글고 부드러운 실루엣, 보송한 섬유 질감, 큰 눈, 은은한 별빛, 따뜻한 시네마틱 조명. 특정 작품을 복제하지 않고 모찌만의 오리지널 정체성을 유지한다.

## 필수 제작 이미지
모든 캐릭터는 다음을 가진다.
- 턴어라운드 5방향
- 표정 10종 이상
- 입 모양 8종 이상
- 액션 포즈 12종 이상
- 호흡/걷기/달리기/점프/착지/시선 이동 사이클
- 한국 장소 상호작용 포즈 : 편의점, 버스정류장, 문구점, 한강, 시장, 학교 앞

## 캐릭터별 고유 효과
- 모찌 : 배 별의 밝기 변화, 뿔의 지연 움직임
- 주무 : 혜성 꼬리, 별가루 흔적, 급정지 잔상
- 기조 : 공구 가방, 톱니 장식, 작은 수리 동작
- 피리 : 감정별 후광, 빛의 떨림
- 포푸 : 구름 몸 변형, 목도리, 작은 비
