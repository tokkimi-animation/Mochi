# Voice replacement required

This render uses the current scratch Korean voice mix only to test timing.

It is not the final voice direction. The final production voices must be created as stable custom character voices:

- Mochi: small, warm, childlike, careful, soft breath, never adult or generic narrator.
- Zumu: bright, quick, playful, slightly airy, childlike, never adult or robotic.

Replacement workflow:

1. Export each Korean line as a separate voice file using ElevenLabs/OpenArt/custom casting.
2. Keep the same start/end timing from `phrase_manifest.csv`.
3. Replace the scratch mix before final YouTube production.
4. Keep light music and SFX under the dialogue so the voices stay clear.

The phrase-matched visual render is valid for story/visual timing review; the voice track is not final.
