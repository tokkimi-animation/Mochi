# EP001 별빛 편의점

러닝타임: 3분. 배경은 한국의 비 오는 밤, 24시간 편의점. 모치와 주무는 길 잃은 별조각들을 겁주지 않고 집으로 돌려보낸다.

- SH001 000.0-008.0s: 비 오는 밤, 별빛 편의점 앞 / 모치가 젖은 길 위에서 편의점 불빛을 발견한다.
- SH002 008.0-016.0s: 간판의 별 / 간판의 작은 별들이 하나씩 따뜻하게 켜진다.
- SH003 016.0-024.0s: 주무의 푸른 꼬리 / 주무가 빗속에서 날아와 모치 옆에 멈춘다.
- SH004 024.0-032.0s: 문 센서의 반짝임 / 자동문 센서가 별가루에 반응하며 빛난다.
- SH005 032.0-042.0s: 안쪽 진열대 / 별조각들이 삼각김밥과 컵라면 사이로 튄다.
- SH006 042.0-052.0s: 미끄러운 바닥 / 모치가 급히 따라가다 멈추고, 주무가 속도를 낮춘다.
- SH007 052.0-062.0s: 불안한 작은 별 / 가장 작은 별조각이 기운을 잃고 낮게 떨린다.
- SH008 062.0-072.0s: 모치의 따뜻한 숨 / 모치가 천천히 숨을 쉬자 가슴 별빛이 부드럽게 번진다.
- SH009 072.0-082.0s: 주무의 약속 / 주무가 꼬리빛을 낮추고 작은 별이 겁먹지 않게 기다린다.
- SH010 082.0-092.0s: 천천히 움직이는 별 / 작은 별이 다시 빛나며 바닥 위로 미끄러진다.
- SH011 092.0-102.0s: 자동문의 박자 / 문이 열리고 닫히는 박자에 별조각들이 줄을 맞춘다.
- SH012 102.0-112.0s: 비와 네온 / 밖의 비와 안의 빛이 유리문 위에서 섞인다.
- SH013 112.0-122.0s: 컵라면 기계 옆 / 마지막 별조각이 뜨거운 물 기계 불빛 뒤에 숨는다.
- SH014 122.0-132.0s: 손바닥 위의 빛 / 모치가 손을 내밀자 별조각이 스스로 내려앉는다.
- SH015 132.0-144.0s: 푸른 길 / 주무가 안전한 푸른 빛길을 만들고 별조각들이 따라간다.
- SH016 144.0-156.0s: 밤하늘로 / 자동문이 열리고 별조각들이 비 사이로 올라간다.
- SH017 156.0-168.0s: 새로 빛나는 간판 / 간판의 별이 조금 더 환해지고 거리의 물웅덩이에 반사된다.
- SH018 168.0-180.0s: 다음 모험의 약속 / 모치와 주무가 웃고 마지막 별가루가 화면을 부드럽게 닫는다.
