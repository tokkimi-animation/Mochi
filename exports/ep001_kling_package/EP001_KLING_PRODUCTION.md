# EP001 Kling production package

Goal: generate the full 3 minute episode through Kling AI as 36 separate 5-second image-to-video clips, then assemble them.

Core rule for every Kling clip:

- Use `VIDEO 3.0`.
- Use the provided start image and end image for each shot.
- Duration: 5 seconds.
- Aspect ratio: 16:9.
- Native Audio: on if available.
- No subtitles, no captions, no readable text added by the model.
- Keep the plush 3D character identity exactly consistent.
- Avoid morphing, melting, extra limbs, separated legs, distorted faces, or camera blur.
- Motion must be small, clear, and character-led: head, eyes, mouth, arms, body bounce, tail glow, floating star dust.
- Lighting must be generated by Kling naturally in the scene, not added as a separate overlay.

Voice direction:

- Mochi: childlike, soft, warm, slightly cautious, Korean voice, not adult.
- Zumu: childlike, airy, bright, faster but gentle, Korean voice, not adult.
- No robotic delivery. Keep character voices consistent across all clips.

Assembly:

- Generate 36 clips of 5 seconds.
- Name them `shot_01.mp4` through `shot_36.mp4`.
- Put downloads into `exports/ep001_kling_package/kling_clips`.
- After all clips are ready, assemble in order into one 3 minute master.

Images:

- Start/end images are in `shot_images`.
- Example: shot 01 uses `shot_01_start.jpg` and `shot_01_end.jpg`.
