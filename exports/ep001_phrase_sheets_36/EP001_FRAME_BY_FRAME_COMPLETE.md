This folder now contains the complete visual basis for episode 1:

- 36 phrase-level source sheets in `source_sheets/`.
- 10 progressive poses per phrase.
- 360 extracted poses generated by `tools/render_ep001_36_phrase_sheets.py`.
- 1800 rendered frames generated by `tools/render_ep001_final_from_360_poses.py`.
- A 3-minute 10fps MP4 preview generated from those frames.

Rules respected:

- No dialogue text is burned into the visual frames.
- Korean subtitles/dialogue must be added only as a final animated overlay.
- Each phrase has a different visual plan and progressive action.
- Everyday Korea references are included through settings and props while preserving the plush character design.
