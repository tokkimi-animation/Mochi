# 모찌의 모험

6세부터 17세까지 함께 볼 수 있는 3분 애니메이션 시리즈 기획/제작 폴더입니다.

## 구성
- `bible/00_시리즈_바이블.md` : 세계관, 톤, 캐릭터, 한국적 배경, 3분 구조
- `bible/01_포즈_표정_매트릭스.md` : 호흡, 표정, 입 모양, 액션 포즈, 바람/빛/감정 연출
- `bible/02_목소리_캐릭터_가이드.md` : 캐릭터별 고정 목소리와 말투
- `production/01_애니메이션_파이프라인.md` : 제작 순서와 품질 기준
- `production/02_캐릭터_비주얼_팩.md` : 캐릭터별 필수 제작 이미지 목록
- `production/03_한국_배경_레퍼런스.md` : 한국에서 벌어지는 느낌을 주는 장소 목록
- `scripts/` : 100편의 3분 에피소드 대본
- `assets/characters/` : 캐릭터 레퍼런스와 제작 시트
- `assets/reference/` : 배경/분위기 레퍼런스

## 핵심 방향
모찌는 한국의 일상 장소와 별빛 판타지가 만나는 세계를 여행합니다. 편의점, 골목, 학교 앞, 버스정류장, 한강 산책로, 문구점 같은 현실적 장소에 문링즈의 빛과 감정 규칙이 스며듭니다.
